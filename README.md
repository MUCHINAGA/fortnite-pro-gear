# フォートナイト世界組デバイスまとめ

Fortniteの世界大会経験者・トッププロを中心に、公開されているデバイス情報を日本語で整理する静的サイトです。

## データ更新

選手データは `data/players.json` で管理します。

ProSettingsの一覧HTMLを取得済みの場合は、次のコマンドで200件のJSONを再生成できます。

```powershell
node scripts/import-prosettings.mjs
```

## 公開

GitHub Pagesで公開できます。詳しい手順は `docs/github-pages-guide.md` を参照してください。
